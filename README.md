# Python 安装

## Python 3.6.2（ http://www.python.org）

Python安装程序可以从www.python.org直接下载，或者也可以直接安装Anacoda Python发行版。

# TensorFlow 安装

## TensorFlow 1.3+ （http://www.tensorflow.org）

安装好Python之后，TensorFlow安装是相当简单的。

`$pip install tensorflow`
